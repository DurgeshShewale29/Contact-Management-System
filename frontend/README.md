# Contact Management App

A simple MERN stack contact management application.

## Features
- Add contact (name, email, phone, message)
- View contacts in real time
- Delete contacts
- MongoDB persistence

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## How to Run
1. Clone the repo
2. Run `npm install` in both frontend and backend
3. Add MongoDB URI in backend `.env`
4. Run `npm start` for backend and `npm run dev` for frontend
